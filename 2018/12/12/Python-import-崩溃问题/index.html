Python import 崩溃问题

  • turtle
  • time
  • numpy

在另一个 Python navigation.py 中，我 import 了另外几个库：

  • numpy
  • skfuzzy
  • from skfuzzy import control
  • matplotlib.pyplot

两个文件分别运行时，都非常正常。但是，我在第一个文件中一旦 import navigation.py 文件（我想使用其中定义的函数），就会收到操作系统抛出的终端异常：
